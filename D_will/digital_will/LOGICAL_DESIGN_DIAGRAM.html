Architecture Overview

The Digital Will DApp uses a hybrid on-chain/off-chain architecture to balance privacy, immutability, and cost-efficiency. Sensitive data is encrypted and stored off-chain, while only cryptographic hashes and references are stored on the blockchain.

🔐 Security Implementation

Encryption & Hashing

  • AES-256-CBC for all sensitive data encryption
  • SHA-256 for will document hashing
  • bcrypt for password hashing with salt
  • Random IVs generated per encryption operation
  • IPFS Web3.Storage for distributed encrypted storage

Access Control

  • Role-based access control (Owner, Beneficiary, Advisor, Admin)
  • JWT authentication for API endpoints
  • Smart contract modifiers for on-chain authorization
  • Encrypted session tokens with expiry

Data Integrity

  • Hash verification on will retrieval
  • Blockchain immutability for transaction records
  • Event logging for comprehensive audit trail
  • IPFS content addressing ensures data hasn't changed
